App React Component

  • App Components
  • App Properties
  • Examples

App React component is the main Framework7 app component where all the Framework7 core initialization happens and where you pass all main Framework7 parameters.

App Components

There are following components included:

  • App / F7App

App Properties

PropTypeDefaultDescription
paramsobjectObject with Framework7 parameters
routesarrayArray with default routes for all views. Same as params.routes so you can use this one if you feel more comfortable with such approach
idstringApp element ID attribute

Examples

Passing all parameters in params prop

import AboutPage from './about.jsx';
import ServicesPage from './services.jsx';

const f7params = {
  name: 'My App',
  id: 'com.myapp.test',
  // routes
  routes: [
    {
      path: '/about/',
      component: AboutPage,
    },
    {
      path: '/services/',
      component: ServicesPage,
    },
  ],
  // ... other params
}
export default () => (
  <App params={f7params}>
    ...
  </App>
);

Passing routes in separate prop:

import routes from './routes';

const f7params = {
  name: 'My App',
  id: 'com.myapp.test',
  // ... other params
};

export default () => (
  <App params={f7params} routes={routes}>
    ...
  </App>
);
