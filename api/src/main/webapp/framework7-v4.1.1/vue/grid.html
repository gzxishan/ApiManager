Grid Vue Component

  • Grid Components
  • Grid Properties
  • Examples

Grid Vue component represents Framework7's Layout Grid.

Grid Components

There are following components included:

  • f7-row - grid row
  • f7-col - grid column (cell)

Grid Properties

PropTypeDefaultDescription
<f7-row> properties
no-gapbooleanfalseRemoves spacing between columns
tagstringdivDefines which tag must be used to render row element
<f7-col> properties
widthnumber
string		autoColumn width. Check available Column Sizes
tablet-widthnumber
string		Column width for large screen tablets (when width >= 768px)
desktop-widthnumber
string		Column width for larger screen tablets (when width >= 1025px)
