Grid Vue Component

    Grid Vue component represents Framework7's Layout Grid.

    Grid Components

    There are following components included:

    • f7-row - grid row
    • f7-col - grid column (cell)

    Grid Properties

    Prop Type Default Description
    <f7-row> properties
    no-gap boolean false Removes spacing between columns
    tag string div Defines which tag must be used to render row element
    <f7-col> properties
    width number
    string    		 auto Column width. Check available Column Sizes
    tablet-width number
    string    		 Column width for large screen tablets (when width >= 768px)
    desktop-width number
    string    		 Column width for larger screen tablets (when width >= 1025px)
